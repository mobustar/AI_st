# 顔認識 (LBPH)

numpy と Pillow だけで動く、軽量な顔認識プログラムです。
PyTorch などの深層学習フレームワークは不要で、1人あたり数枚の画像でも認識できます。

## 仕組み

**LBPH（局所二値パターン・ヒストグラム）** という手法を使っています。

1. 画像をグレースケール化し、固定サイズ（100x100）にそろえる
2. 各画素を周囲8近傍と比較し、明暗パターンを 0〜255 のコードに変換（LBP）
3. 画像を 8x8 のセルに分割し、各セルのヒストグラムを連結して特徴ベクトルにする
4. 認識時は、登録済み画像の特徴と最も近いものを探す（カイ二乗距離による最近傍）

照明変化に強く、少ない枚数でも安定するのが特長です。

## 必要なモジュール

```bash
pip install numpy pillow
```

## ファイル構成

```
face_recognition/
├── lbph_core.py    共通処理（画像読み込み・LBP特徴抽出）
├── train.py        学習 → face_model.npz を生成
├── recognize.py    face_model.npz を使って認識
├── README.md       このファイル
└── dataset/        学習用の画像を置くフォルダ
```

## 使い方

### 1. 学習用画像を配置する

`dataset/` の下に、人ごとのフォルダを作って画像を入れます。
フォルダ名がそのまま人物名（ラベル）になります。1人あたり1〜3枚でも動きます。

```
dataset/
├── alice/
│   ├── a1.jpg
│   └── a2.jpg
└── bob/
    └── b1.jpg
```

対応形式: `.jpg` `.jpeg` `.png` `.bmp`

> 精度を上げるコツ: 画像は **顔だけを切り抜いて** 置くと認識精度が上がります。

### 2. 学習ファイルを生成する

```bash
cd face_recognition
python train.py
```

成功すると `face_model.npz` が作られます。これが学習済みデータ（外部ファイル）です。
一度作れば、認識側はこのファイルを読み込むだけで動きます（再学習は不要）。

### 3. 顔を認識する

認識したい画像を指定して実行します。

```bash
python recognize.py test.jpg
```

出力例:

```
判定: alice  (距離 0.123)
```

距離が小さいほど似ています。登録外の人の場合は次のように表示されます。

```
判定: 未登録の人 (最も近いのは alice, 距離 0.712)
```

## 調整できるパラメータ

| ファイル | 変数 | 説明 |
|---|---|---|
| `recognize.py` | `THRESHOLD` | この距離より遠いと「未登録の人」と判定。小さくすると厳しめ（既定 0.5） |
| `lbph_core.py` | `SIZE` | 画像の統一サイズ（既定 100x100） |
| `lbph_core.py` | `GRID` | ヒストグラムの分割数（既定 8x8）。細かくすると位置に敏感になる |

## 補足

- 学習結果は `np.savez` で `.npz` ファイルに保存しているため、PyTorch などは不要です。
- 新しい人を追加したいときは `dataset/` に画像を足して `python train.py` を再実行してください。
